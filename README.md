tkSIInformer
============

Информер журнала "Самиздат" на python/ttk


Зависимости
===========

python 3
tkinter/ttk


Первый запуск
=============

Сначала нужно инициализировать базу данных
    $ ./main.py --init
Эта команда создаст в локальном каталоге файл data.sqlite. Далее запускаем:
    $ ./main.py
